# Red Hat Single Sign-On Workshop

Red Hat Single Sign-On (RH-SSO) is based on the Keycloak project and enables you to secure your web applications by providing Single Sign-On (SSO) capabilities. It supports widely adopted standards such as OpenID Connect, OAuth 2.0, and SAML 2.0, allowing seamless authentication and authorization for your applications.

---
NOTE: This repository is archived as it represents the final version of the product under this name. It is considered historical and will no longer be actively maintained. For workshops or tutorials related to the current product, please refer to the updated repository (Red Hat build of Keycloak).
